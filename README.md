#  Classic Bites | Restaurant Menu Website

 is a modern and responsive restaurant menu webpage built using pure HTML, CSS, and JavaScript  
It showcases a restaurant’s menu with a search bar, category filters, and attractive design.



##  Features

-  Instantly find dishes as you type.  
-  Filter menu items by category — *Starters, Main Course, Desserts, Drinks  
-  Works smoothly on desktop and mobile screens.  
-  Elegant typography, hover effects, and card layout.  
-  100% vanilla HTML, CSS, and JS — no external dependencies.
